# Internal Docs

The Orivon Foundation has its own [Internal Documentation](https://orivonbrowser.github.io/internal-docs) to help contributors orientate in the project

Here an index to better orientate into it:

- **If you want to be a Technical Contributor, [Developers Work](https://orivonbrowser.github.io/internal-docs/docs/developer-work) and [Technical Philosophy](https://orivonbrowser.github.io/internal-docs/docs/technical-philosophy) are a *must read***

- **If you want to become Involved in the Orivon Foundation, [Involvement](https://orivonbrowser.github.io/internal-docs/docs/involvement) is a *must read***

- **If you want free guides and resources made for Orivon contributors access [Resources](https://orivonbrowser.github.io/internal-docs/docs/category/resources)**
