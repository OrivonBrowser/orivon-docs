# orivon-core

*orivon-core* is the official Orivon client wrapping *orivon-runtime* which uses it to actual implement the backend technology behind Orivon Browser

It implements and guarantees that the official *Orivon standards* are safely and accurately implemented and enforced when needed

*orivon-core* will be probable directly component that's going to be embedded inside the Orivon Browser
